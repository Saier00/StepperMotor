Программируемый блок управления контроллером шагового двигателя BL-TB6560-V2.0 управляется терминалом через COM-порт ПК и имеет два режима работы:
-Ручное управление (DRC)
-Выполнение заданного алгоритма (ALG)

В режиме "Ручного управления" выполнение команд осуществляется в режиме реального времени.
В режиме выполнения алгоритма, набор команд единожды считывается с ПК, а команды выполняются последовательно.
//Сохранение и считывание алгоритма из памяти на данный момент не реализованно
